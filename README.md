What is this?

This is my first project, a very simple one, yet im proud of it. It checks file types and organizes large amounts of files into different folders, Saving you the time and effort for mass amounts of files.




How does this work?

1. Check the "Messy" Folder. It's super unorganized right!!
2. Forget about the hassle of making your own folders and doing it yourself! (Useful for hugee amounts of files)
3. Open file_organizer.py
4. Run it, it will ask for your directory (give the directory for "Messy" for example)
5. Wait a moment, then check the "Messy" folder again!
6. Sit back and relax knowing you didn't have to do a thing!


How does the code work?

Lines 4-12: It is saying "here is a folder" "these files go in here" , example: here is a folder : "Images" , These files go in here ".jpg", ".jpeg", ".png", ".gif", ".bmp", ".tiff". If anything ends with these file names, they will automatically go into the images folder. (Feel free to add file names to fit your needs)

Lines 16-20: Defines Organize_Files, checks if the directory is valid and if not it returns "Error:'{directory}' is not a valid directory."

The rest of the program has comments in file_organizer.py to guide you!
